<코드 테스트 방법 안내 - Game_Upgrade.ipynb 코드 파일>
   1. 실행 환경 준비
      - 이 코드를 Jupyter Notebook이나 IPython 환경에 복사하여 실행하세요.
      - Python 환경에서 pygame, gTTS, speech_recognition 등의 라이브러리가 설치되어 있어야 합니다.

   2. 코드 실행 및 기능 동작
      1) 코드 실행 시, 먼저 전체 문장("난독증 아이들을 위한 게임입니다")이 화면에 표시됩니다.
      2) 한 글자씩 TTS 재생 및 시각적 강조
         - 각 글자가 순차적으로 화면에 나타나며, 현재 읽는 글자는 #ff8e03 색상으로 강조되고 나머지 글자는 기본 색상(#45a049)으로 표시됩니다.
         - 각 글자에 대해 gTTS를 통해 생성된 음성이 재생됩니다.
      3) 전체 문장 TTS 재생
         - 한 글자씩 읽은 후, 전체 문장이 노란색으로 크게 표시되며 TTS 음성이 재생됩니다.
         - 이때 기존에 생성된 sentence.mp3 파일이 있으면 삭제하고 새로 생성됩니다.
      4) 사용자 녹음 및 비교
         - 화면에 표시된 문장을 따라 사용자가 읽으면, 마이크로 녹음이 시작됩니다.
         - 녹음된 음성은 Speaking 폴더에 WAV 파일로 저장되며, HTML 오디오 플레이어를 통해 재생 컨트롤이 화면에 표시됩니다.
         - 저장된 녹음 음성은 Google 음성 인식 API를 통해 텍스트로 변환되어 원본 문장과 비교됩니다.
      5) 차이점 강조 표시
         - 만약 인식된 문장이 원본과 일치하지 않으면,
             > 예상 문장과 인식된 문장의 차이를 SequenceMatcher를 사용해 분석합니다.
             > 예상 문장에서는 틀린 부분을 빨간색(연한 분홍색 배경)으로, 인식된 문장에서는 틀린 부분을 파란색(연한 하늘색 배경)으로 강조하여 단락 형식으로 화면에 표시됩니다.
         - 이후 "틀린 부분을 확인하고 다시 시도해주세요."라는 메시지가 출력되며, 최대 3회까지 시도할 수 있습니다.

   3.  결과 확인
         - 정상적으로 진행되면 "테스트 통과!"라는 메시지가 출력됩니다.
         - 테스트 실패 시, 차이점이 강조된 문장을 참고하여 어떤 부분이 인식되지 않았거나 다르게 인식되었는지 확인할 수 있습니다.
